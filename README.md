# Student Grade Tracker

This is a simple Python command-line project that manages student data and performs basic analysis.

## Features

- Cleans raw student data (names, roll numbers, marks)
- Validates names using string checks
- Assigns grades based on marks
- Calculates total, average, highest and lowest scores
- Generates a class performance report
- Performs string operations on a sample essay

## Concepts Used

- Loops (for, while)
- Conditional statements (if-else)
- Lists and tuples
- String methods
- Exception handling (try-except)

## How to Run

1. Open terminal in project folder
2. Run:

python part1_grade_tracker.py

3. Enter input when prompted
